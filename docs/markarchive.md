# MarkArchive
> nothing is everything

MarkArchive is an open-source organization focused on hardware and software (making tools and systems that are usually proprietary accessible to independent developers and small teams)

If you want to know more: [github.com/MarkArchive](https://github.com/MarkArchive)

---

## Why is this project under MarkArchive?
Häte5d engine is a MarkArchive project because game engines are critical infrastructure that should not be locked behind proprietary tools or licenses. Häte is fully open, auditable, and extensible by anyone.
